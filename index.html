Цимайло Ветошкина Березина

При изучении этого языка мы обратили внимание, что использование подобных функций недостаточно распространено среди фронтенд-разработчиков

Поэтому мы решили изучить необычные функции CSS и узнать, насколько они популярны

CSS — что это?

› Инструмент для веб-разработки

CSS (Сascading Stylesheets) — язык для определения того, как документы представляются пользователям — как они стилизованы, оформлены и т.д.

C его помощью веб-разработчики придают базовым элементам HTML уникальный дизайн. Например,можно задать тексту цвет или расположить изображение. Постепенно CSS развивается: на данный момент последней версией является CSS3, которая постоянно обновляется.

Современные тренды

Упрощение Многофункциональность

Развитие CSS приводит к тому, что он упрощает жизнь разработчикам введением более удобных функций. Кроме того, вводятся методы, которые позволяют создавать эффекты ранее недоступные веб-разработчикам.

Переменные CSS

Дата введения:
2017 г.
Осведомлены:
98.7% опрошенных
Используют:
93% опрошенных

Переменные позволяют быстро адаптировать дизайн при изменении каких‑либо параметров. Например, если базовые цвета вашего макета поменялись и вы назначали их через переменные, вам достаточно их изменить в переменных, а не менять во всех элементах.

.div {
  background-color: blue;
}
.span {
  background-color: blue;
}
:root {
  --main-color: blue;
}
.div {
  background-color: var(--main-color);
}

.span {
  background-color: var(--main-color);
}

Нестандартное использование переменных расширяет возможности: например интерполяция (переход) градиентов. Задавая тип данных для переменной, мы даем браузеру понять, что конкретно требуется поменять в сложнообрабатываемых функциях. В случае градиентов — это цвет, проценты градиента, его угол и т.д.

@property --x {
  syntax: '';
  inherits: false;
  initial-value: 10%;
}
@property --y {
  syntax: '';
  inherits: false;
  initial-value: 10%;
}
box {
  transition: --x 0.2s, --y 0.2s;
  background: radial-gradient(
    circle at var(--x) var(--y),
    blue 20%,
    white 30%,
    blue 50%
    &);
}

87% опрошенных

С помощью этой функции эффект Paralaxa теперь достигается даже без JS. Paralax - это явление, при котором разные слои одного и того же изображения двигаются с разной скоростью, создавая иллюзию глубины. Важно, что теперь мы можем анимировать наши объекты непосредственно при прокрутке, используя scroll() и view().

Тригонометрические функции

Дата введения:
2023 г.
Осведомлены:
80% опрошенных
Используют:
10% опрошенных

На данный момент CSS умеет обрабатывать 8 тригонометрических функций: sin(), cos(), tan(), asin(), acos(), atan(), and atan2(). С их помощью можно описывать движения практически любого характера, так как мы можем эффективно обращаться к углу поворота и т.д. Одним из базовых, но наглядных примеров является вращение объекта по кругу.

x = cos(a) * r
y = sin(a) * r
translateX: calc(cos((var(--angle) + var(--angle-offset))) * var(--radius))
translateY: calc(sin((var(--angle) + var(--angle-offset))) * var(--radius) * -1);

Вложения CSS

Дата введения:
2021 г.
Осведомлены:
88% опрошенных
Используют:
27% опрошенных

Вложения и нововведенные селекторы, как :has(), :where(), :is() позволяют упрощать процесс написания кода. Они просто минимизируют повторения и делают код более структурным.

.parent1 .child1,
.parent2 .child1 {
  &color: red;
}
.parent1 .child2,
.parent2 .child2 {
  &color: green;
}
.parent1 .child2:hover,
.parent2 .child2:hover {
  &color: blue;
}
.parent1, .parent2 {
  .child1 {
    color: red;
  }
  .child2 {
    color: green;
    &:hover {
      color: blue;
    }
  }
}
